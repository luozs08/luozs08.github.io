FFmpeg常用的几条命令

FFmpeg是一套可以用来记录、转换数字音频、视频，并能将其转化为流的开源计算机程序 。 它十分强大，可能为了保持稳定性，它没有图形界面，只能用命令实现。下面介绍几中常用的命令。

合并视频

ffmpeg -i "concat:input1.mts|input2.mts|input3.mts" -c copy output.mts 这是无损合并的方式，我用它来合成相机录制的视频。

压缩视频

ffmpeg -i input.mp4 -b:v 500k -s 854x480 output.mp4 这是将视频分辨率从1920x1080转成854x480的命令，其中500k是常用的码率。

截取视频

ffmpeg -ss 00:01:00 -t 00:15:47 -accurate_seek -i input.mp4 -codec copy output.mp4 这是从1分钟开始截取长度为15分47秒视频的命令。
